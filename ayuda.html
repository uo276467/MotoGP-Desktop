Ayuda de MotoGP-Desktop

Bienvenido al manual de ayuda de MotoGP-Desktop. Aquí encontrarás explicaciones y recomendaciones para utilizar todas las secciones de la aplicación web, así como soluciones a problemas habituales y enlaces a los recursos técnicos del proyecto.

Inicio

La página de inicio ("Inicio") ofrece un resumen con acceso rápido a las distintas secciones: Piloto, Circuito, Meteorología, Clasificaciones y Juegos. Utiliza los enlaces del menú para navegar por la aplicación. La página muestra un carrusel de imágenes del Misano World Circuit y noticias relacionadas con MotoGP.

Piloto

En la sección "Piloto" se muestra información sobre el piloto Pedro Acosta (biografía, datos personales, trayectoria y logros). La página incluye un glosario con palabras propias del mundo de Motogp

Circuito

La página "Circuito" muestra información geográfica y técnica del circuito (mapas, longitudes, curvas y altimetría).

Uso y recursos:

  • Para ver información general del circuito, introduce el archivo InfoCircuito.html de la carpeta xml
  • Para visualizar la altrimetría del circuito, introduce el archivo altimetria.svg de la carpeta xml
  • Para visualizar el recorrido del circuito, introduce el archivo circuito.kml de la carpeta xml

Meteorología

La sección "Meteorología" ofrece pronósticos y datos del tiempo para la ubicación del circuito.

Clasificaciones

"Clasificaciones" es una página que muestra el podio de la última carrera de MotoGP celebrada en Misano World Circuit

Juegos

En "Juegos" se agrupan mini-aplicaciones:

  • Juego de memoria con cartas: un minujuego donde el usuario deberá levantar cartas y enontrar parejas con logos relacionados con MotoGP.
  • Cronómetro: un cronómetro implementado en javascript.
  • Cronómetro PHP: un cronómetro implementado en PHP.
  • Configuración test: contiene funciones para manipular la base de datos de las pruebas de usabilidad.
  • Test Usabilidad: un cuestionario que sirve de prueba para evaluar la usabilidad del sitio web.
